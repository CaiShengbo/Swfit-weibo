# 用swift 写的weibo

这个是我在学习Swift过程中利用业余时间写的一个很简单程序,代码都是照着oc写的,只是为了学习swift嘛,其实在学习swift中要学习的还很多，尤其对extention的使用和协议变成的理解。

感谢 [YYKit提供强大功能的支持]

### 图片效果演示

![image](https://github.com/agelessman/Swfit-weibo/raw/master/gif_weibo.gif)

### 包含功能

* 各种富文本的使用展示
* 微博首页的展示
* Twitter的展示
* 图片的异步加载展示



### Requirement

* iOS 8.0+ 
* Xcode 7.0 +
